## How to Build
Here's how to build my dookie code.
1. Install Visual Studio
    * I used [VS2019](https://aka.ms/vs/16/release/vs_community.exe) but you should be able to use [VS2022](https://visualstudio.microsoft.com/)

2. Get the Project
    * To get the project, use
    ```
    git clone https://github.com/MonkeTGG/.NETaskbar
    cd .NETaskbar
    explorer .
    ```
3. Open in Visual Studio
    * Double click the file.

4. Now let's build!
    * Click `Build > Build Solution` on the topbar.

5. Find it on your PC.
    * Should be `${downloadLocation}\bin\Debug\.NETaskbar.exe`
